## Setup

1. `cd backend && cp .env.example .env`
2. Fill in your Supabase and Fanvue credentials in `.env`
3. `npm install`
4. Apply `schema/schema.sql` and `schema/schema_v2.sql` to your Supabase database
5. `npm start` (or `node server.js`)
6. Open `http://localhost:5000` in your browser
